Conversions between JSON and biniou
val biniou_of_json : Yojson.Safe.json -> Bi_io.tree
Converts from JSON to biniou. Raises Failure if conversion is not reversible.
val json_of_biniou : Bi_io.tree -> Yojson.Safe.json
Converts from biniou to JSON. Raises Failure if conversion is not reversible, except for biniou tables which are considered equivalent to arrays of records.
