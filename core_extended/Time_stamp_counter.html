module Cycles : sig .. end
Small routines for working with the timestamp counter. The snapshot is machine specific information that is calculated once during a run and cached, though the first time it is requested it may take several 100 milliseconds. Cycle information captured on one machine can be translated to Core.Time.t on another machine only if the a snapshot of the first machine is also available
